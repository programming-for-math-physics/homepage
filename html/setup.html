Pythonを使ってみる

起動

ここでは起動のテストだけを行う. もっと便利な方法は後に.
  • Linux: 端末からコマンドを入力するのが基本. 画面左のランチャ「端末」 を開き, コマンドプロンプトにpythonと入力 
    • $ python
Python 2.7.3 (default, Sep 26 2013, 20:03:06) 
[GCC 4.6.3] on linux2
Type "help", "copyright", "credits" or "license" for more information.
>>>
  • Mac持参の場合: 端末からコマンドを入力するのが基本. Finder -> アプリケーション -> 端末 (ドックに入れておくとよい) を開き, コマンドプロンプトにpythonと入力
  • Windows持参の場合: スタート -> 全てのプログラム -> Python 2.7 -> Python IDLE (Python GUI)

式の入力

  • >>>がPythonのプロンプト(入力待ち)
  • プロンプトに向かって, 1+1, 1+2*3, 3**100 など, 色々な式を打ち込んでみよ (下線部分が入力する部分). 
    >>> 1+1
>>> 1+2*3
>>> 3**100

終了

プロンプトに向かって, Ctrl-d を入力すると終了する

Visual Pythonを使って見る

Visual Pythonの起動

  • Linux: Visual PythonはPythonの機能を拡張するもので, 起動すべきコマンド自身はPythonと同じである. Pythonを立ち上げた後以下が無事実行できればOK. 
    >>> from visual import *
>>> sphere()
    2つ目を入力した際に以下が表示されれば成功.
  • Mac: Linux同様の手順でうまく行くはずなのだが, 何故かうまくいかない. とりあえず以下の手順で
    1. Finder -> アプリケーション -> VIDLE 2.7 for Python とする
    2. そこに, 以下を打ち込む. 
      from visual import *
sphere()
      これはPythonに打ち込んでいるのではないので, 打ち込んだだけでは何もおこらない.
    3. その上で, メニューから, Run -> Run Module (または F5 キーをおす)と, Pythonが立ち上がり, 入力した内容がPythonに送られ, 無事上の絵が出るはず.
  • Windows: Linux同様の手順でうまく行くはずなのだが, 何故かうまくいかない. とりあえず以下の手順で
    1. スタートメニュー -> VIDLE for Python とすると, (Pythonのプロンプトではなく)ファイルを編集するエディタが立ち上がる
    2. そこに, 以下を打ち込む. 
      from visual import *
sphere()
      これはPythonに打ち込んでいるのではないので, 打ち込んだだけでは何もおこらない.
    3. その上で, メニューから, Run -> Run Module (または F5 キーをおす)と, Pythonが立ち上がり, 入力した内容がPythonに送られ, 無事上の絵が出るはず.

Visual Pythonの終了

Visual Pythonの窓が出ている状態では, PythonにCtrl-d を送っても終了しない. 窓を閉じてから, Ctrl-dで終了.

エディタでプログラムを編集・保存する

直接Pythonのプロンプトに式を打ち込むやり方では, Pythonを立ち上げる度に毎度同じ事を打ち込む必要がある. ある程度以上長いプログラムはファイルに保存してそれを実行するのが基本である. プログラムを記録したファイルを編集するには, エディタと呼ばれるツールを用いる. エディタには何種類もある. 普段から使い慣れたエディタがある人はそれを使ってもよい. ただし, 以下の手順に従うと, プログラムの編集から実行までがスムースに行えるのでおすすめする.

gedit

一番簡易なエディタ． 画面左上のUbuntuメニューをクリックし，ge くらいまで うちこめば(geditを途中までタイプすれば)，自然に選択肢の中に， 「テキストエディタ」というのが現れるのでそれを選ぶ．
まずはややこしいことを一切覚えたくないという人は， これを使うと良い． ただし以下のEmacsを使いこなせるようになったほうが， 後々効率があがる．

Emacsエディタ

  • Linux : 左側のランチャから起動
  • Mac : Finder -> アプリケーション -> Emacs (ドックに入れておくとよい)
  • Windows: スタート -> 全てのプログラム -> Meadow

Emacsエディタの基本操作

  • メニューからひとしきりのことを行うことができる
  • ただし, よく使うものはキーボードで行えるようにすると良い. メニューの中に対応するキーボード操作が書かれている
    • ファイルを開く・作る: C-x C-f
    • 保存: C-x C-s
    もちろんもう少し色々なコマンドを覚える必要がある. それはおいおい覚えておけば良い. こちらに初歩的なコマンド集を用意した.

EmacsエディタでPythonプログラムを編集・保存

Pythonプログラムは, .py という拡張子で保存する. Emacsを用いて, a.py というファイルを作り, 以下の内容で保存し, 再び開いてみよ (どのフォルダに保存しているのかだけ, 意識できるように). 
from visual import *
sphere()

Pythonのより便利な立ち上げ方

  1. Linux, Mac : 端末を立ち上げ, 保存済みファイルのあるディレクトリに行き, 
    $ python ファイル名
  2. Windows : 保存したファイルをダブルクリックする
  3. Linux : EmacsでPythonファイルを開いている状態で,
    • Emacsのメニュー -> Python -> Start Interpreter とすると, Emacs エディタ内で Pythonが立ち上がる. 同じ操作は C-z でもできる.
    • そこへ向かって普通に式などを入力しても良い
    • Pythonファイルのバッファにカーソルがある状態で, Emacsのメニュー -> Python -> Eval Buffer とすると, そのファイルの内容が, Emacs エディタ内で立ち上がったPythonへ送られる. 同じ操作は C-c C-c でもできる. プログラムの編集 -> テスト -> 編集 -> テスト, というサイクルを素早く回すことができるので慣れることをすすめる.
    • Mac : Linuxと同じことができるはずなのだがなぜか絵が表示されない. 当面は1の方法で行うしかない(ようだ). これは, Visual Pythonを 用いる際にのみ現れる問題で, 通常のPythonの使用においては問題ない.

Matplotlibを使って見る

プログラムの準備

Matplotlibはデータの可視化(グラフ化)を得意とする.
  • 以下のようなプログラム(例: mp.py)をエディタで準備 (以下をコピペしても良い, これをダウンロードしても良い) 
    import matplotlib.pyplot as plt
import math

X = range(0,100)
Y = [ math.sin(0.2 * x) for x in X ]

plt.plot(X, Y)
plt.show()
  • 端末から以下のコマンドで実行する 
    $ python mp.py
  • こんなグラフ(y = sin(x))があらわれれば成功

numpyを使って見る

プログラムの準備

numpyは数値計算全般を得意とする. 最もわかりやすいご利益は以下のように, 行列やベクトルの演算が簡単に行えること.
  • 以下のようなプログラム(例: np.py)をエディタで準備 (以下をコピペしても良い, これをダウンロードしても良い) 
    import numpy as np
A = np.matrix([[1,2],[3,4]])
x = np.matrix([[5],[6]])
y = A * x
print A
print x
print y
# Ax = y を解く
print np.linalg.solve(A, y)
  • 実行して以下のように表示されれば成功 
    $ python np.py
[[ 5.]
 [ 6.]]

scipyを使って見る

プログラムの準備

scipyも数値計算全般を得意とし, numpyよりも機能が拡張されている. 例えば以下は関数の最小を求める(最適化) 機能を用いて, 円に内接する三角形の面積の最大値 (正確にはそれをSとして-2Sの最小値)を求めている.
  • 以下のようなプログラム(例: sp.py)をエディタで準備 (以下をコピペしても良い, これをダウンロードしても良い) 
    import scipy.optimize
from math import pi,sin

def f(x):
    return - sin(x[0]) - sin(x[1]) - sin(2.0 * pi - x[0] - x[1])

print scipy.optimize.minimize(f, (0.5, 0.5))
  • 実行して以下のような結果が表示されれば成功.
